## 5. 当前建议优先级

如果只抓“最重要的东西”，建议按下面顺序推进：

1. 先补 `Observation` 的 `list / info / create / update`
2. 再补聚合接口：
   - `dataset -> links`
   - `material -> datasets`
   - `assay -> datasets`
3. 然后推进 phenotype 文件结构化导入
4. 最后再接前端入口

原因：

- 现在缺的不是更多单表 CRUD
- 而是把 breeding 主体、关联层和 phenotype 主线真正串起来
