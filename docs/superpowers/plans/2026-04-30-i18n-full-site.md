# 全站多语言 (i18n) 支持实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为全站约 1100+ 处硬编码中文创建 locale key，启用语言切换器，支持 zh-CN ↔ en-US 即时切换。

**Architecture:** 启用已有的 LanguageToggle widget（改一个默认值），创建 12 个 namespace 的 locale JSON 文件（zh-CN + en-US 各一份），在约 119 个源文件中将硬编码中文替换为 `$t('namespace.key')` 调用。不引入新依赖，不修改框架层。

**Tech Stack:** vue-i18n (Composition API), `$t` from `@vben/locales`, ant-design-vue ConfigProvider

- [ ] **Step 3: 修改 workspace/index.vue**

在 `<script setup>` 顶部添加 `import { $t } from '@vben/locales';`

然后将所有硬编码中文替换为 `$t('workspace.*')` 调用。有三种模式：

**computed（参数化）：**
```typescript
const workspaceDescription = computed(() => {
  const teamName = projectStore.getTeamInfo?.team_name?.trim?.();
  const projectName = projectStore.getProjectInfo?.name?.trim?.();
  if (teamName && projectName) return $t('workspace.currentTeamProject', { team: teamName, project: projectName });
  if (teamName) return $t('workspace.currentTeam', { name: teamName });
  if (projectName) return $t('workspace.currentProject', { name: projectName });
  return $t('workspace.fallbackDesc');
});
```

**欢迎消息（ref 初始化）：**
```typescript
const chatMessages = ref<WorkspaceChatMessage[]>([{
  id: 'welcome', role: 'assistant',
  content: $t('workspace.chat.welcome'),
}]);
```

**脚本 error 消息：**
```typescript
// noModel
content: $t('workspace.error.noModel'),
// HTTP error
$t('workspace.error.requestFailed', { code: response.status, message: errorText || $t('workspace.error.unknown') }),
// Abort
$t('workspace.chat.cancelled'),
// Catch-all
$t('workspace.error.streamFailed', { message: error?.message || $t('workspace.error.unknown') }),
```

**Template 文本：** 全部 `文本` → `{{ $t('workspace.xxx') }}`，key 对应 JSON 中的路径。
